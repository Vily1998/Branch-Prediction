# Branch-Prediction
分支预测（Branch Prediction）是现代处理器用来提高CPU执行速度的一种手段, 其对程序的分支流程进行预测, 然后预先读取其中一个分支的指令并解码来减少等待译码器的时间。

A strategy in computer architecture design for mitigating the costs usually associated with conditional branches, particularly branches to short sections of code.


Our model：Tage predictor + bimodal predictor + Loop predictor + clock
